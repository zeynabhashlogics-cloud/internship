## Installation

Clone the repository

```bash
git clone https://github.com/your-username/javascript-practice.git
```

Go into the project folder

```bash
cd javascript-practice
```

Install dependencies

```bash
npm install
```

---

## Running a Program

Example:

```bash
node calculator.js
```

or

```bash
node swap.js
```
